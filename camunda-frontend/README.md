# Camunda Frontend

React frontend application for managing Camunda workflow tasks.

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will start on http://localhost:3000

## Environment Variables

Create a `.env` file in the root directory:

```
REACT_APP_CAMUNDA_API_URL=http://localhost:5050/api/camunda
```

## Features

- View all active tasks
- Filter tasks by candidate group
- Claim and unclaim tasks
- Complete tasks with form data
- View process instances
- View process variables
